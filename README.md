## Primeros pasos

Se debe crear la base de datos manualmente
- Nombre = prueba
- usuario = postgres
- clave = postgres

## Archivo de configuraci√≥n .env
- DB_CONNECTION=pgsql
- DB_HOST=localhost
- DB_PORT=5432
- DB_DATABASE=prueba
- DB_USERNAME=postgres
